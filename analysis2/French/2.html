Applications en Informatique

1. Continuité et Dérivées : L'Optimisation 🧠

En apprentissage automatique (Machine Learning), l'entraînement d'un modèle consiste à minimiser une "fonction de coût" qui mesure son erreur. Pour trouver le minimum de cette fonction, on utilise des algorithmes d'optimisation comme la descente de gradient. Cet algorithme repose entièrement sur le calcul de la dérivée (le gradient) de la fonction de coût. La dérivée indique la direction de la plus forte pente ; en allant dans la direction opposée, on minimise l'erreur. La notion de continuité est cruciale car ces méthodes ne fonctionnent que sur des fonctions "lisses" (continues et dérivables).

Exemple Pratique : Ajustement d'un Modèle Linéaire

Imaginez qu'on veuille trouver la meilleure droite (y = ax + b) qui passe par un nuage de points. La fonction de coût pourrait être la somme des carrés des distances verticales entre chaque point et la droite. Pour trouver les meilleures valeurs de 'a' et 'b', on calcule les dérivées partielles de cette fonction de coût par rapport à 'a' et 'b' et on les ajuste itérativement pour faire tendre ces dérivées vers zéro.

parameters = { a: random(), b: random() }
learning_rate = 0.01

function train(data) {
  for (step in 1..1000) {
    // Calculer la dérivée (gradient) de la fonction de coût
    grad_a = cost_derivative_wrt_a(data, parameters)
    grad_b = cost_derivative_wrt_b(data, parameters)

    // Mettre à jour les paramètres dans la direction opposée au gradient
    parameters.a -= learning_rate * grad_a
    parameters.b -= learning_rate * grad_b
  }
}

2. Développements Limités : L'Approximation Efficace ⚡️

Beaucoup de fonctions mathématiques (sin, cos, exp, log...) sont coûteuses à calculer pour un ordinateur. Les développements limités (ou séries de Taylor) permettent d'obtenir une excellente approximation de ces fonctions en utilisant uniquement des polynômes (additions et multiplications), qui sont extrêmement rapides à calculer. En informatique, on n'a souvent pas besoin de la valeur exacte, mais d'une approximation suffisamment bonne. C'est le fondement de nombreuses bibliothèques scientifiques et de calcul.

Exemple Pratique : Calcul de sin(x)

Plutôt que d'utiliser une méthode complexe, une calculatrice ou un programme peut utiliser les premiers termes du développement limité de sin(x) autour de 0 pour obtenir un résultat très précis pour des petites valeurs de x. L'équation de la tangente est simplement le développement limité à l'ordre 1.

// DL de sin(x) = x - x^3/6 + x^5/120 - ...

function fast_sin(x) {
  // Pour x proche de 0, on peut se contenter des premiers termes
  let x_squared = x * x;
  let x_cubed = x_squared * x;
  let x_quinte = x_cubed * x_squared;

  // Approximation à l'ordre 5
  return x - (x_cubed / 6.0) + (x_quinte / 120.0);
}
